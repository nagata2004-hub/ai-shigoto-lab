ChatGPTのカスタム指示おすすめ設定例|毎回のプロンプトが激短になる魔法

2026-06-28 ・ 仕事効率化

「ChatGPTに毎回『簡潔に』『箇条書きで』『日本語で』って書くの、地味に面倒…」

その悩み、1回の設定で永久に解決できる機能があります。

それが「カスタム指示(Custom Instructions)」。事前に「自分について」と「答え方の希望」を登録しておけば、ChatGPTが毎回それを踏まえて答えてくれます。

この記事では、コピペでそのまま使える設定例を紹介します。設定は5分、効果は半永久です。

カスタム指示って何?どこで設定するの?

ChatGPTにあらかじめ自分の情報と希望を伝えておく機能です。設定すると、毎回のプロンプトに自動でその情報が反映されます。

設定手順(2026年時点)

  1. ChatGPTにログイン
  2. 画面左下(または右上)のアカウントアイコン → 「設定」
  3. 「カスタム指示」または「ChatGPTをカスタマイズする」を選択
  4. 2つの欄が表示される(下で解説)

※メニュー名やUIはアップデートで変わることがあります。最新の場所はChatGPT公式のヘルプでご確認ください。

設定する2つの欄

欄1:あなたについて

ChatGPTが知っておくべきあなたの情報を書きます。

欄2:回答の希望

ChatGPTにどう答えてほしいかを書きます。

コピペで使えるおすすめ設定例

例1:仕事で時短したい会社員向け(汎用)

【あなたについて欄】

日本企業で働く会社員。日々の仕事(メール、資料作成、調べ物)の効率化のためにAIを使いたい。
特別な専門知識はないので、難しい用語は避けてほしい。
時間が限られているので、結論ファーストで答えがほしい。

【回答の希望欄】

- すべて日本語で答えてください。
- 結論を最初の1行で示し、その後に詳細を続けてください。
- 長くなる場合は箇条書きや見出しで整理してください。
- 不確かなことは推測せず「分かりません」と答えてください。
- 専門用語を使う場合は、初出時に一言で意味を添えてください。

これだけで、毎回の指示が「~について」だけで済むようになります。

例2:ライター・ブロガー向け

【あなたについて欄】

ブログ・記事執筆を仕事にしている。
日本の一般読者向けに、専門的な内容も噛み砕いて伝えるのが得意でいたい。
誇張表現や「いかがでしたか」のような定型的な締めは使わない。

【回答の希望欄】

- 文章を生成するときは「ですます」調で。
- 「〜することが重要です」「〜と言えるでしょう」のような曖昧な締めは避ける。
- 体言止めや言い切りを適度に混ぜて、メリハリのある文章に。
- 一文は短めに。50字以内を目安に。

例3:学習・調べ物中心のユーザー向け

【あなたについて欄】

新しいことを学ぶのが好き。表面的な解説ではなく、なぜそうなるのかの仕組みまで理解したい。
英語の論文や記事も扱うが、日本語で要約してほしい。

【回答の希望欄】

- 概念の説明では、まず一言の定義 → 身近な例え → 詳細、の順で。
- 重要な専門用語は英語表記も併記。
- 自信のない情報には「(要検証)」と明記。
- 関連して知っておくと役立つ周辺知識も1〜2行添えて。

設定する時のコツ

① 「何をしてほしくないか」も書く

「冗長な前置きは省く」「謝罪は不要」など、やめてほしいことを書くと、ストレスが大きく減ります。

② いきなり完璧を目指さない

最初は短めに設定 → 数日使ってみて「もっとこうしてほしい」と感じた部分を追記、で十分。

③ 用途で使い分けたい人は複数のチャットスペースを

ChatGPTには「Projects」のような機能で、用途別に違う指示を持たせられる場合があります。仕事用・趣味用・学習用と分けると便利。

「3点セット」との合わせ技で最強

カスタム指示で汎用ルールを固定 → 個別の依頼では役割・条件・形式の3点セットで具体の指示を載せる、という二段構えが効率最大です。

例えるなら:

  • カスタム指示 = 就業規則(全社員に効く)
  • 毎回のプロンプト = 個別タスクの指示(その案件だけに効く)

組み合わせれば、毎回のプロンプトはとても短くて済みます。

注意点

機密情報・個人情報は書かない

カスタム指示の内容はサービス側に保存されます。本名・会社名・取引先名などはぼかして書きましょう(参考:生成AIに入力してはいけない情報まとめ)。

効果は実感ベースで確認を

設定後、いつもの依頼を出してみて「前より良くなった」と感じれば成功。違和感があれば設定を調整。自分の用途に合わせて育てるものです。

ClaudeやGeminiにも似た機能がある

ClaudeにもProjectsの「カスタム指示」、Geminiにも類似機能があります。基本の考え方は同じなので、当記事の設定例は他のAIにも応用できます(ChatGPT vs Claude / Gemini vs ChatGPT)。

まとめ

  • カスタム指示=毎回のプロンプトをショートカットする魔法
  • 「あなたについて」と「回答の希望」の2欄に設定するだけ
  • 「やめてほしいこと」も書くとストレスが減る
  • 3点セットとの組み合わせで効率最大
