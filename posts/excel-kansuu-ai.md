「Excelでこういう計算をしたいけど、どの関数を使えばいいか分からない」

そんな時、もう関数を暗記する必要はありません。**やりたいことを日本語で伝えるだけで、AIが関数を作ってくれます。**

この記事では、ChatGPTやClaudeにExcelの関数を作ってもらう手順を、初心者向けに解説します。

## 基本の頼み方

AIに関数を作ってもらうとき、伝えるべきことはシンプルです。

1. **どのセルに何が入っているか**(データの配置)
2. **何をしたいか**(やりたい計算・処理)
3. **結果をどこに出したいか**

この3つを伝えるだけ。難しい専門用語は不要です。

### コピペで使える頼み方の型

```
Excelの関数を教えてください。
・A列に商品名、B列に単価、C列に個数が入っています。
・D列に「単価×個数」の合計金額を出したいです。
・D2セルに入れる関数を教えてください。
```

これで、AIが `=B2*C2` のような関数と、その意味を説明してくれます。

## よくある「やりたいこと」別の頼み方

### 条件で表示を変えたい(IF関数)

```
B列の点数が80以上なら「合格」、それ未満なら「不合格」とC列に表示する関数を教えてください。
```

### 別の表からデータを探して持ってきたい(VLOOKUP / XLOOKUP)

```
「商品番号」をもとに、別シートの一覧表から「商品名」を自動で表示したいです。関数を教えてください。シート名やセル範囲も含めて説明してください。
```

### 特定の条件に合うものだけ合計したい(SUMIF)

```
A列が「東京」の行だけ、B列の売上を合計する関数を教えてください。
```

### 重複や空白を整理したい

```
A列にある重複したデータを除いて、ユニークな一覧を作る方法を教えてください。
```

## エラーが出た時の頼み方

関数を入れて `#N/A` や `#VALUE!` などのエラーが出ても、あわてず**そのままAIに聞けばOK**です。

```
この関数を入れたら #N/A というエラーが出ました。原因と直し方を教えてください。
関数:【ここに貼り付け】
```

エラーの種類から原因を推測して、修正方法を教えてくれます。

## 精度を上げるコツ

うまくいかない時は、**実際のデータの形をもっと具体的に伝える**のが効果的です。

- 「1行目は見出しで、データは2行目から始まります」
- 「日付はB列に2026/6/18のような形式で入っています」

こうした“前提”を渡すほど、AIの答えは正確になります。この考え方は[AIプロンプトの基本「役割・条件・形式」の3点セット](prompt-kihon-3ten.html)で詳しく解説しています。思った答えが出ない時は[AIの回答が的外れな時の直し方5パターン](ai-kaitou-naoshikata.html)も参考にしてください。

## 注意:結果は必ず自分で確認する

AIが作った関数は、**簡単なテストデータで動作を確認**してから本番に使いましょう。特に、

- 計算結果が手計算と合っているか
- セルの範囲(A2:A100など)が自分の表と合っているか

この2点だけは目視で確認する習慣を。また、社外秘の数字をそのまま入力しない点にも注意してください(参考:[生成AIに入力してはいけない情報まとめ](ai-nyuryoku-kikenjouhou.html))。

## まとめ

- Excel関数は暗記不要。「データの配置・やりたいこと・出力先」を伝えればAIが作る
- IF・VLOOKUP・SUMIFなどもやりたいことを日本語で言うだけ
- エラーもそのまま貼って質問すれば直してくれる
- 結果は簡単なテストで必ず確認する

関数を調べて何十分も悩む時代は終わりました。次にExcelで手が止まったら、まずAIに聞いてみてください。
