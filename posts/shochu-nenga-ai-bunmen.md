暑中見舞いや年賀状——毎年「去年何て書いたっけ」と悩みながら、同じような文面を考え直していませんか。

**この文面作成は、AIが最も得意とする作業の一つです。** 相手との関係と伝えたい一言を伝えるだけで、数秒で自然な文面ができあがります。

この記事では、暑中見舞い・年賀状の文面をAIに作ってもらう手順と、そのまま使えるプロンプト例をまとめました。

## 基本の頼み方

伝えるべきことはシンプルです。

1. **相手との関係**(取引先/上司/友人/親戚 など)
2. **一言添えたい内容**(近況、感謝、次に会いたい旨 など)
3. **文章の長さ・トーン**(ハガキ1枚分、堅め/柔らかめ)

### コピペで使える型

```
暑中見舞いの文面を作ってください。
・相手:【取引先の担当者様】
・関係:【今年お世話になった方】
・伝えたい内容:【今後ともよろしくお願いしたい旨】
・トーン:【丁寧だが堅すぎない】
・長さ:【ハガキ1枚に収まる程度】
```

これで、季節の挨拶から結びの言葉まで、そのまま使える文面が出てきます。

## シーン別プロンプト例

### 取引先・ビジネス関係(暑中見舞い)

```
取引先への暑中見舞いの文面を作成してください。日頃の感謝と、酷暑の中での健康を気遣う一文を入れてください。丁寧なビジネス文書のトーンで。
```

### 上司・目上の方へ

```
お世話になっている上司への暑中見舞いを作成してください。堅苦しくなりすぎず、日頃の指導への感謝が伝わる文面にしてください。
```

### 友人・親戚へ(カジュアル)

```
親しい友人への暑中見舞いを、カジュアルで温かみのある文面で作成してください。近況を尋ねる一言も入れてください。
```

### 年賀状(ビジネス)

```
取引先への年賀状の文面を作成してください。昨年のお礼と、本年も変わらぬお付き合いをお願いする内容で、フォーマルなトーンにしてください。
```

### 年賀状(親戚・友人向け)

```
親戚への年賀状を、家族の近況を軽く触れつつ、温かみのある文面で作成してください。
```

頼み方の基本的な考え方は[プロンプトの基本「役割・条件・形式」の3点セット](prompt-kihon-3ten.html)と同じです。

## 一段上げるコツ

### ① 出来事を1つだけ加える

```
今年、部署異動があったことにも軽く触れた文面にしてください。
```

**個人的な出来事を1つ入れる**だけで、定型文からぐっと自然な手紙になります。

### ② 複数パターン出してもらって選ぶ

```
上記の内容で、少しずつトーンの違う文面を3パターン作ってください。
```

比較して一番しっくりくるものを選べます。

### ③ 一言だけ自分の言葉を足す

AIの文面をそのまま使うより、**最後に一文だけ手書き風の一言を自分で加える**と、グッと心が伝わる文面になります。

## 注意点

### ① 送るタイミングのマナーは自分で確認
暑中見舞い・残暑見舞い・年賀状には、それぞれ送るべき時期のマナーがあります。時期の慣習は年によって解釈が変わることもあるため、不安な場合は最新のマナー情報を別途確認するのが安全です。

### ② 相手の個人情報は入力しない
「取引先の◯◯様」のように**名前や会社名は入力せず、後から自分で書き加える**のが安全です(参考:[生成AIに入力してはいけない情報まとめ](ai-nyuryoku-kikenjouhou.html))。

### ③ 事実と違う内容が混ざらないよう確認
「昨年は大変お世話になり」のような定型句が、実際の関係と合っているか、送る前に読み返しましょう。

## まとめ

- 「相手との関係・伝えたい内容・トーン」の3点を伝えるだけで文面が完成
- ビジネス・友人・親戚など、シーン別に頼み方を変えるだけで自然な仕上がりに
- 出来事を1つ加える、一言手書き風に足すと、より心のこもった文面になる
- 送るタイミングや個人情報の扱いには注意

今年の暑中見舞いから、まずAIに下書きを頼んでみてください。悩む時間がなくなった分、相手のことを思い浮かべる時間に使えます。
