メール1通に10分、20分と悩んでいませんか。

文面を考える時間は、AIに任せればほぼゼロになります。この記事では、**コピペして【 】の中を書き換えるだけで使える**ビジネスメールのプロンプトを12個紹介します。

## 使い方は3ステップ

1. 下のプロンプトをコピーする
2. 【 】の部分を自分の状況に書き換える
3. ChatGPTやClaudeに貼り付けて送信する

出てきた文面を確認して、少し手直しすれば完成です。慣れると1通あたり1〜2分で送れるようになります。

## 基本の型(これだけ覚えればOK)

すべてのプロンプトに共通する「効くコツ」は次の3つです。

- **相手との関係**を伝える(社外/上司/初めての相手 など)
- **伝えたい結論**を先に渡す(AIに考えさせない)
- **トーン**を指定する(丁寧に/簡潔に/柔らかく など)

この3点は、メールに限らずあらゆるAIへの指示で効きます。詳しくは[AIプロンプトの基本「役割・条件・形式」の3点セット](prompt-kihon-3ten.html)で解説しているので、あわせてどうぞ。まずは下のテンプレを使ってみてください。

## シーン別プロンプト12例

### 1. 依頼メール

```
あなたはビジネスメールの専門家です。以下の内容で、社外の取引先に送る依頼メールを作成してください。トーンは丁寧かつ簡潔に。
・依頼内容:【見積書の送付】
・期限:【今週金曜まで】
・補足:【急ぎで申し訳ない旨を添える】
```

### 2. お礼メール

```
以下の内容で、お礼のビジネスメールを作成してください。堅すぎず、感謝が伝わる自然な文面で。
・相手:【先日打ち合わせした取引先】
・お礼の内容:【貴重な時間と資料の提供】
```

### 3. お詫び・謝罪メール

```
以下の内容で、社外向けのお詫びメールを作成してください。誠実で、言い訳がましくならないトーンで。
・お詫びの内容:【納品が1日遅れたこと】
・今後の対応:【再発防止のため確認体制を強化する旨】
```

### 4. 催促メール(角を立てずに)

```
以下の内容で、相手に不快感を与えない催促メールを作成してください。柔らかく、確認のお願いという体裁で。
・催促する内容:【先週依頼した資料の進捗】
・補足:【相手が忙しい可能性に配慮する】
```

### 5. 日程調整メール

```
以下の内容で、打ち合わせの日程調整メールを作成してください。相手が返信しやすいよう候補を提示する形で。
・目的:【新サービスの説明】
・候補日:【来週の火・水・木の午後】
```

### 6. 断り・辞退メール

```
以下の内容で、相手との関係を壊さない断りのメールを作成してください。感謝を示しつつ、明確に辞退する形で。
・断る内容:【今回の見積もり提案】
・理由:【社内の都合により今回は見送り】
```

### 7. 初めての相手への挨拶メール

```
初めて連絡する社外の相手へのメールを作成してください。失礼のない丁寧なトーンで、簡単な自己紹介を含めて。
・自分:【○○社の営業担当】
・用件:【製品カタログ送付の許可をいただきたい】
```

### 8〜12. その他よく使う場面

残りはプロンプトの「依頼内容」を変えるだけで対応できます。

- **8. 問い合わせへの回答**:「以下の問い合わせに、丁寧に回答するメールを作成してください」
- **9. 見積もり提出**:「見積もりを提出する案内メールを作成してください」
- **10. リマインド**:「明日の会議のリマインドメールを、簡潔に作成してください」
- **11. 退職・異動の挨拶**:「お世話になった社外の方への挨拶メールを作成してください」
- **12. 返信が遅れたお詫び**:「返信が遅れたことを詫びつつ本題に入るメールを作成してください」

## 使うときの注意

- **送信前に必ず読み返す**:宛名・日付・金額・固有名詞はAIが間違えることがあります。ここだけは人間が確認を。
- **機密情報の入力に注意**:取引先名や個人情報をそのまま入力してよいか、会社のルールを確認しましょう(関連:議事録の記事でも触れています → [会議の議事録作成をAIで自動化する方法](ai-gijiroku-jidouka.html))。
- **自分の言葉を少し足す**:全文AIのままだと味気ないので、一言だけ自分の言葉を加えると、ぐっと自然になります。

## まとめ

- メール作成は「関係・結論・トーン」の3点を伝えるだけでAIに任せられる
- シーン別テンプレを保存しておけば、1通1〜2分で送れる
- 送信前のチェック(宛名・数字・機密情報)だけは人間の仕事

毎日メールを書く人ほど効果が大きい使い方です。ちなみに「無料版でも足りるか、有料版が必要か」が気になる方は、[ChatGPT有料版は元が取れる?](chatgpt-yuryoban-moto.html)もあわせてどうぞ。
