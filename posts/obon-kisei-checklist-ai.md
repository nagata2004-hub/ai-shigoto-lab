お盆休みの帰省、荷造りのたびに「あれ、去年何を忘れて困ったんだっけ」と思い出せず、結局その場で考え直す——という人は多いはずです。

**この「毎年ゼロから考える」作業こそ、AIにチェックリスト化してもらうのが向いています。** この記事では、実際に使えるプロンプト例と、活用のコツをまとめます。

## 基本の頼み方

伝えるべき条件はシンプルです。

1. **移動手段**(車/新幹線/飛行機)
2. **同行者**(大人だけ/子ども連れ/高齢の親を訪ねるなど)
3. **滞在日数**
4. **特別な予定**(お墓参り、親戚の集まりなど)

### コピペで使える型

```
お盆の帰省準備のチェックリストを作ってください。
・移動手段:【新幹線】
・同行者:【夫婦と未就学児1人】
・滞在日数:【3泊4日】
・予定:【実家でお墓参りあり】

持ち物・事前にやっておくこと(郵便物・冷蔵庫の整理など)に分けて、
カテゴリごとにチェックリスト形式で出してください。
```

この伝え方は、[プロンプトの基本「役割・条件・形式」の3点セット](prompt-kihon-3ten.html)の考え方そのままです。条件を具体的に伝えるほど、実用的なリストが返ってきます。

## シーン別に頼むとさらに精度が上がる

### 子ども連れの場合

```
未就学児を連れての帰省です。移動中と実家滞在中、それぞれで必要な子ども用品のチェックリストを作ってください。
```

### 車移動の場合

```
車で帰省します。出発前の車のチェック項目(タイヤ・ETC・車検証など)もリストに含めてください。
```

### 高齢の親を訪ねる場合

```
高齢の両親の家に帰省します。健康状態を確認する際に気をつけたい観点や、持っていくと喜ばれるものの候補も提案してください。
```

### 家を空ける前の確認事項

```
3泊4日、自宅を空けます。忘れがちな「家を空ける前にやること」のチェックリストを作ってください(郵便物、冷蔵庫、電気・ガス、防犯など)。
```

## 実際に使ってみて便利だった点

- **「持ち物」と「やること」を分けて整理してくれる**ので、頭の中がごちゃごちゃにならない
- **子ども連れ・車移動など、状況に応じたリストにカスタマイズできる**
- できあがったリストをそのままメモアプリにコピーして、**チェックしながら荷造りできる**

## 注意点

### ① 交通機関・道路状況は必ず最新情報を確認
お盆時期の混雑予測や高速道路の渋滞情報は、AIの知識では最新の状況を反映できません。出発前に必ず交通情報サイトや公式アプリで確認しましょう。

### ② 実家・親戚の個人情報は入力しない
「〇〇県の実家」のように**具体的な住所や家族の個人情報は入力せず**、一般的な条件で頼むのが安全です(参考:[生成AIに入力してはいけない情報まとめ](ai-nyuryoku-kikenjouhou.html))。

### ③ リストは「たたき台」として扱う
家庭ごとに必要なものは異なります。AIのリストをベースに、**自分の家庭に合わせて追加・削除する**のを忘れずに。

## まとめ

- 移動手段・同行者・日数・予定を伝えるだけで、帰省準備のチェックリストが作れる
- 子ども連れ・車移動・高齢の親訪問など、シーン別に頼むとさらに実用的に
- 交通情報は最新のものを別途確認、個人情報は入力しない
